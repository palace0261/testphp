사용자가 원하는데로 커스텀

전용 소프트웨어를 통해 멀티 버튼과 노브의 기능을 사용자가 원하는 대로 설정할 수 있습니다. 또한 CAM1~10버튼을 원하는데로 이름을 변경하여 보다 직관적으로 사용할 수 있습니다

소스 선택

Blackmagic Design 스위처 전제품호환 조작가능하며 ATEM 컨트롤 패널의 직관적인 물리 버튼을 통해 보다 안정적인 스워처 운용이 가능합니다.

최대 20채널 지원

"SHIFT 키"를 통해 ATEM 2 M/E Constellation의 20개의 채널 PGM/PVW 버튼을 빠르게 전환할 수 있습니다 또한 키매핑 기능을 통해 입력소스 40채널까지 변경해서 사용할수 있습니다.
ATEM 2 M/E Constellation HD/4K

멀티 버튼

패널 상단의 7개의 멀티 기능 버튼을 통해 자주 사용하는 기능으로 (Aux, Aux Output, M/E) 설정할 수 있습니다. (*SHIFT 키와 조합하여 최대 14개의 멀티 기능 버튼 사용 가능)

IP제어로 간단사용

ATEM 스위처의 모든 제품을 LAN 케이블 연결을 통해 스위처의 기능을 컨트롤 조작할 수 있습니다.


예시)
[스위처] - 블랙매직 ATEM 스위처 IP 주소 : 192.168.10.240

[컨트롤 패널] - SMC-50 설정
1) 이더넷 IP 주소 : 192.168.10.100
2) ATEM IP 주소 : 192.168.10.240(ATEM 스위처 IP와 동일하게 세팅바랍니다)

트랜지션 제어

MIX, DIP, WIPE, STING, DVE등 각 트랜지션 유형을 선택하여 다양한 효과를 실행할 수 있습니다.

업스트림 키

크로마 키, 루마 키, 등 배경이 되는 소스 위에 다른 영상이나 그래픽을 오버레이하는 방식으로 버튼을 눌러 간편하게 전환할 수 있습니다.

다운스트림 키

다운스트림 키(DSK) 기능을 통해 로고, 그래픽 등을 영상 위에 오버레이할 수 있습니다.

오디오 컨트롤

오디오, XLR, MIC, MP 등 멀티 채널 오디오 제어 기능을 지원합니다.

페이더 바

9단계 LED 라이트 프로그레스 바가 결합되어 수준 높은 방송이나 공연에 수동으로 제어하여 출연진과 호흡을 바탕으로 완벽한 타이밍에 맞춰 사용 가능합니다.

페이드 투 블랙(FTB)

지정된 속도로 서서히 어두워지며 모든 방송이 종료되거나 쉬어갈 때 혹은 갑작스러운 방송 사고를 대비하여 유용하게 사용 가능한 버튼입니다.

Power over Ethernet

PoE 기능을 지원하여 네트워크 허브를 통해 별도의 전원입력 없이 LAN 케이블만으로 전원공급이 가능합니다.

인터페이스

LAN 케이블 연결만으로 ATEM 스위처의 모든 기능을 손쉽게 물리적인 버튼을 통해 빠르고 정화하게 제어할 수 있습니다.
