KBD PLUS

KBD PLUS는 PTZ 카메라 제어와 영상 모니터링을 하나의 장치에서 통합적으로 수행할 수 있는 혁신적인 컨트롤러입니다.
More than a PTZ Controller
  • 비디오 디코딩
    최대 4개의 IP 비디오 스트림을 동시에 낮은 지연 시간으로 디코딩하여 실시간 모니터링 및 제어가 가능합니다.
  • 멀티뷰 모니터링
    여러 대의 카메라 영상을 하나의 멀티뷰 레이아웃에서 편리하게 확인하고 제어하면서 모니터링할 수 있습니다.
  • HDMI Out
    HDMI 2.0 출력은 디코딩된 카메라 영상을 단일 화면 또는 다중 화면 레이아웃으로 외부 모니터에 표시합니다.
  • USB-C Out
    USB-C 출력은 UVC를 지원하므로 디코딩된 비디오가 소프트웨어 애플리케이션에서 웹캠으로 표시될 수 있습니다.
    • 실시간으로 최대 4채널의 IP 비디오 스트림을 디코딩하여 멀티뷰로 제공하며, HDMI 및 USB 출력을 지원하여 다양한 디스플레이와 녹화 장치에 연결할 수 있습니다. 직관적인 인터페이스와 강력한 기능으로 방송, 라이브 이벤트, 원격 회의 등 다양한 영상 제작 환경에서 최적의 솔루션을 제공합니다.
    AV Over IP

    • IP Video Streaming

    IP 네트워크를 통해 고품질 비디오 스트리밍을 지원하여 원격 모니터링과 제어가 가능합니다. 이를 통해 사용자는 어디서든 카메라의 영상을 실시간으로 확인하고 관리할 수 있습니다.

  • Open API Integration

  • 외부 장치에 API 명령을 전송하여 사용자 지정 제어
    • Where Control, Visibility, and Intelligence Come Together

    • 정밀도와 내구성을 고려하여 제작된 맞춤형 조이스틱

    KBD Plus 조이스틱은 전문가용 PTZ 카메라 조작에 특화되어 설계되었으며, 부드러운 반응, 정밀한 위치 지정 및 뛰어난 내구성을 제공합니다. 일반적인 조이스틱 모듈과 달리, 전문가용 카메라 조작에 최적화된 설계를 자랑합니다.
    ▶ 100만 회 이상의 동작 주기 테스트를 거쳐 장기적인 내구성을 보장합니다.
    ▶ 50만 회 회전 후에도 0.7%의 오차만 발생하는 정밀한 제어 정확도를 유지합니다.
    ▶ 70만 회 이상의 실제 제어 동작을 지원하도록 설계되어 긴 작동 수명을 자랑합니다.
    Control
  • 터치스크린
    반응 속도가 뛰어난 4인치 터치스크린을 통해 카메라 제어, 모니터링 및 시스템 설정에 빠르게 접근할 수 있습니다.
  • 카메라 선택
    미리보기를 통해 각 카메라가 촬영하는 화면을 확인할 수 있어 원하는 카메라를 쉽고 빠르게 찾을 수 있습니다.
  • 프리셋 설정
    각 카메라의 사전 설정 위치에 대한 썸네일 미리보기를 통해 원하는 사전 설정을 쉽게 빠르게 식별하고 불러올 수 있습니다.
  • 키패드, 원버튼
    키패드 직접 호출 또는 번호 입력 후 호출 명령을 사용하여 카메라 또는 사전 설정된 카메라를 즉시 불러올 수 있습니다.

    • 다양한 활용도

    천장, 벽, 기둥, 트라이포드(삼각대)등 다양하게 설치가 가능하며 우수한 영상 출력으로 교회, 학원, 사무실, 라이브, 스포츠, 화상회의, 병원 등 다양한 환경에서 최적의 방송 비디오 영상 출력 스트리밍이 가능합니다.
