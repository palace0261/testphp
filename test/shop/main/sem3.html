ATEM Mini Pro

빠르게 셋업하고 쉽게 사용할 수 있는 HDMI 스위처. 다양한 입력/출력, 오디오 믹서, 크로마 키, 멀티뷰, 스트리밍 및 녹화 기능을 제공합니다.

주요 사양

비디오 입력 4 x HDMI (10비트 HD, 2채널 임베디드 오디오, 재동기화/포맷 컨버터 지원)
비디오 출력 1 x HDMI 프로그램 출력
오디오 입력 2 x 3.5mm 스테레오 미니 잭
오디오 출력 임베디드 오디오만 지원
USB 인터페이스 1 x USB 타입 C 3.1 (외장 드라이브 녹화, 웹캠 출력 등)
이더넷 10/100 BaseT (제어, 스트리밍, 패널 연결)
지원 해상도/포맷
  • 입력: 720p50/59.94/60, 1080p23.98~60, 1080i50/59.94/60
  • 출력/스트리밍: 1080p23.98~60
  • UHD: 미지원
색 정밀도/샘플링 10비트, 4:2:2 YUV, Rec 709
멀티뷰 1 x 10분할(입력 4, 미디어, 프리뷰, 프로그램, 상태 등)
미디어 플레이어 1 (스틸 이미지 20개, PNG/TGA/BMP/GIF/JPEG/TIFF 지원)
오디오 믹서 6입력 x 2채널, Fairlight EQ/컴프/게이트/리미터, 마스터 게인
스트리밍 이더넷 RTMP 직접 스트리밍 지원
녹화 USB-C 외장 미디어에 H.264 .mp4 직접 녹화
제어 내장/외장 패널, ATEM Software Control (Mac/Windows)
전원 12V DC 어댑터 (30W)
크기/무게 소형 데스크탑 폼팩터
작동 환경 5°~40°C, 0~90% 비응결
운영체제 지원 Mac 14.0+, Windows 10/11
보증 1년 한정 보증

특징 요약

  • 4개의 HDMI 입력, 표준 변환 및 재동기화 지원
  • USB-C 웹캠 출력 및 외장 저장장치 녹화
  • 이더넷을 통한 직접 스트리밍(RTMP)
  • 멀티뷰, 오디오 믹서(Fairlight), 크로마 키, DVE, 다양한 트랜지션
  • 20개 스틸 이미지 미디어 풀
  • 무료 ATEM Software Control 제공
